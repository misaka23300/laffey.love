# embedded IDE配置

大家好，这节课介绍一个ai集成开发环境.传统的单片机程序开发门槛较高，AI编程工具的出现正在逐渐改变这个情况。通过使 用自然语言描述需求，AI可自动生成底层驱动代码、优化外设配置，甚至可协助完成实时 调试。2025年，字节跳动公司基于开源项目，推出AI原生IDE-Trae，Trae国内版预置了豆包、DeepSeek等大模型，提供Builder和chat两种工作 模式。Builder 模式可以帮助开发者从零开始构建项目，Chat模式支持对代码库或编程问题 进行提问和优化



TRAE 与 AI 深度集成，提供智能问答、代码自动补全以及基于智能体的 AI 自动编程能力。使用 TRAE 开发项目时，你可以与 AI 灵活协作，提升开发效率。

完备的 IDE 功能

提供传统的 IDE 功能，包括代码编写、项目管理、插件管理、源代码管理等。

强大的 AI 助手

- **智能问答**

- 在编写代码时，你可以随时与 AI 助手对话，获得代码解释、代码注释和错误修复等帮助。详情参考

侧边对话和

- 行内对话。

- **实时代码建议**

- AI 助手能够理解当前代码并在编辑器中实时提供代码建议，提升编程效率。详情参考

- CUE。

- **代码片段生成**

- 通过自然语言描述你的需求，AI 助手将生成相应的代码片段，甚至能够编写项目级或跨文件的代码。

- **从 0 到 1 开发项目**

- 告诉 AI 助手你想开发的程序，AI 助手将根据你的描述提供相关代码或自动创建所需文件。详情参考

- 智能体。

SOLO 模式

SOLO 模式是一种高度自动化的开发方式，以 AI 为主导，自动规划并执行从需求理解、代码生成、测试，到成果预览的全流程。目前，SOLO 模式已开放等待名单，前往 [TRAE CN 官网](https://www.trae.cn/solo)进行预约。

支持的操作系统

- macOS 10.15 及以上

- Windows 10、11

## 设置工具链 

我们使用keil5对代码进行编译，所以首先设置工具链，找到keil5的文件位置。

我们点击`操作`中的`设置工具链`,

![QQ_1761189668313](./embedded%20IDE%E9%A1%B9%E7%9B%AE%E9%85%8D%E7%BD%AE.assets/QQ_1761189668313.png)

点击keil C51，来选择keil5文件目录中uv4.exe的文件位置。

![QQ_1761189731787](./embedded%20IDE%E9%A1%B9%E7%9B%AE%E9%85%8D%E7%BD%AE.assets/QQ_1761189731787.png)

## 设置中文

选择好后，我们可以配置插件的语言为中文。

![QQ_1761189781266](./embedded%20IDE%E9%A1%B9%E7%9B%AE%E9%85%8D%E7%BD%AE.assets/QQ_1761189781266.png)

我们点击`操作`中的`打开插件设置`，在设置中往下拉，找到`EIDE：Display Language`,然后来选择我们熟悉的中文吧。

接下来我们新建项目。我们使用是keil5编译的51单片机。

我们点击`新建项目`,选择`空项目`，选择`8位MCU项目`，选择`Empty Project`,
